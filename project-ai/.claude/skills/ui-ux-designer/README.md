# UI/UX Designer Skill

A skill that helps design beautiful UI components following your established design system.

## Purpose

This skill assists in creating beautiful UI components by generating HTML/CSS code that adheres to your specific design system requirements.

## Knowledge Base

The skill incorporates knowledge of:

- Your color palette (primary blue #2563eb, secondary purple #9333ea, and neutral colors)
- Your spacing system (8px base unit with multiples: 8px, 16px, 24px, 32px, 48px, 64px)
- Accessibility standards to ensure components are usable by everyone
- Your typography scale for consistent text hierarchy
- Component patterns like cards, buttons, navigation, etc.

## Capabilities

- Generate semantic HTML structures
- Create CSS that follows your design system specifications
- Implement proper spacing using your 8px grid system
- Apply your color palette appropriately
- Follow your typography scale for visual hierarchy
- Include accessibility attributes and ARIA roles where needed
- Ensure responsive design principles

## Usage Examples

- "Create a button component following my design system"
- "Design a card component with proper spacing and colors"
- "Generate a navigation component with accessibility features"
- "Create a form element with proper styling and accessibility"
