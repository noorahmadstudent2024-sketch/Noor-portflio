# Browsing with Playwright Skill

A skill that enables automated web browsing and testing using Microsoft's Playwright framework.

## Purpose

This skill provides automated browser interactions, page navigation, element inspection, and functional testing capabilities for web applications using Microsoft's Playwright framework. It supports Chromium, Firefox, and WebKit browsers with a single API.

## Knowledge Base

The skill incorporates knowledge of:

- Playwright API for browser automation
- Cross-browser testing (Chromium, Firefox, WebKit)
- Element selection and interaction
- Page navigation and routing
- Network interception and mocking
- Screenshots and PDF generation
- Form filling and submission
- JavaScript execution in browser context
- Waiting strategies and synchronization
- Headless and headed browser modes

## Capabilities

- Launch and control browsers programmatically
- Navigate to URLs and interact with pages
- Select and manipulate DOM elements
- Fill forms and submit data
- Take screenshots and generate PDFs
- Intercept and mock network requests
- Execute JavaScript in browser context
- Handle popups, dialogs, and modals
- Wait for elements and conditions
- Perform cross-browser testing
- Record and replay user interactions
- Debug and inspect web pages programmatically

## Usage Examples

- "Open Chrome and navigate to my portfolio"
- "Take a screenshot of the projects page"
- "Fill out the contact form with test data"
- "Test the navigation on my website"
- "Simulate user interactions with my UI"
- "Verify that buttons are clickable"
- "Check if my site loads correctly in Firefox"
