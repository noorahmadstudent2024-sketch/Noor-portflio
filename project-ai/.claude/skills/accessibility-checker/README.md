# Accessibility Checker Skill

A skill that helps evaluate and improve web accessibility compliance for your portfolio website.

## Purpose

This skill provides comprehensive accessibility checking for your portfolio website, helping ensure it meets WCAG (Web Content Accessibility Guidelines) standards and is usable by people with disabilities.

## Knowledge Base

The skill incorporates knowledge of:

- WCAG 2.1 AA guidelines
- ARIA (Accessible Rich Internet Applications) standards
- Keyboard navigation requirements
- Screen reader compatibility
- Color contrast ratios
- Alternative text for images
- Form accessibility
- Semantic HTML structure
- Focus management
- Cognitive accessibility considerations

## Capabilities

- Audit pages for accessibility violations
- Check color contrast ratios against WCAG standards
- Identify missing alternative text for images
- Verify proper heading structure
- Assess form accessibility and labeling
- Evaluate keyboard navigation flow
- Recommend ARIA attributes where needed
- Identify potential screen reader issues
- Suggest improvements for cognitive accessibility
- Validate semantic HTML usage
- Check focus indicators and management

## Usage Examples

- "Check accessibility of this page"
- "Audit my navigation for accessibility"
- "Review my form for accessibility issues"
- "Test color contrast in my design"
- "Evaluate my images for alt text compliance"
- "Analyze my site for keyboard navigation"
