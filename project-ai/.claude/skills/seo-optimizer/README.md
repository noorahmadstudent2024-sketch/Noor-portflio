# SEO Optimizer Skill

A skill that helps optimize your portfolio website for search engines.

## Purpose

This skill provides comprehensive SEO optimization assistance for your portfolio website, helping improve visibility in search engine results and attract more visitors.

## Knowledge Base

The skill incorporates knowledge of:

- On-page SEO best practices
- Meta tags optimization (title, description, keywords)
- Header structure and content hierarchy
- Image optimization (alt tags, file names)
- Internal linking strategies
- Page speed optimization
- Mobile responsiveness
- Accessibility standards (which also impact SEO)
- Schema markup for rich snippets

## Capabilities

- Analyze content for SEO effectiveness
- Suggest optimal meta tags for pages
- Recommend header structure improvements
- Provide image optimization suggestions
- Advise on internal linking opportunities
- Identify duplicate content issues
- Suggest keyword placement and density
- Recommend schema markup for portfolio items
- Provide accessibility improvements that boost SEO
- Suggest performance optimizations

## Usage Examples

- "Optimize my homepage for SEO"
- "Review this page for SEO improvements"
- "Suggest meta tags for my portfolio page"
- "How can I improve SEO for my projects section?"
- "Analyze this content for search engine visibility"
