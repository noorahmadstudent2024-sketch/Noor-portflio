# Portfolio Website

A modern, minimalist portfolio website built with HTML, CSS, and JavaScript to showcase web development projects.

## Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling with modern layout techniques (Flexbox/Grid)
- **JavaScript (ES6+)** - Interactive elements and functionality

## Features

- **Modern Design** - Clean, minimalist aesthetic with thoughtful typography
- **Responsive Layout** - Mobile-first approach ensuring optimal viewing on all devices
- **Dark Mode Support** - Toggle between light and dark themes based on user preference
- **Project Showcase** - Dedicated sections to highlight key projects with descriptions
- **Smooth Animations** - Subtle transitions for enhanced user experience
- **Accessibility** - Semantic HTML and proper ARIA attributes for screen readers
- **Fast Loading** - Optimized assets and efficient code for quick loading times

## Project Goals

- Create a professional online presence to showcase development skills
- Demonstrate proficiency with HTML, CSS, and JavaScript
- Implement modern UI/UX principles
- Provide an easy way for potential clients/employers to browse work
- Highlight key projects with case studies and outcomes

## Target Audience

- Potential employers looking for front-end developers
- Clients seeking web development services
- Fellow developers interested in collaboration
- Personal branding and networking

## Key Sections

- **Hero Section** - Introduction and brief overview
- **About Me** - Background and skills
- **Projects** - Portfolio of work with live demos
- **Skills** - Technical competencies
- **Contact** - Ways to get in touch

## Design Philosophy

Minimalist design approach focusing on content and user experience. Clean typography, ample whitespace, and intuitive navigation to ensure projects are the primary focus. Dark mode support for comfortable viewing in different lighting conditions.

## Design System

### Color Palette
- **Primary Blue**: #2563eb (used for primary buttons, links, and key interactive elements)
- **Secondary Purple**: #9333ea (used for accents, highlights, and secondary elements)
- **Neutral Colors**:
  - Light theme: White backgrounds, gray text (#1f2937)
  - Dark theme: Dark backgrounds (#111827), light text (#f9fafb)

### Spacing Scale
- **Base Unit**: 8px
- **Scale**: 1unit (8px), 2units (16px), 3units (24px), 4units (32px), 6units (48px), 8units (64px)
- **Usage**: Consistent spacing between elements, padding, and margins following the 8px grid system

### Typography Scale
- **Display**: 2.5rem (40px) - Hero titles
- **Heading 1**: 2rem (32px) - Main section headers
- **Heading 2**: 1.5rem (24px) - Subsection headers
- **Heading 3**: 1.25rem (20px) - Card titles
- **Body Large**: 1.125rem (18px) - Lead paragraphs
- **Body Regular**: 1rem (16px) - Standard text
- **Body Small**: 0.875rem (14px) - Captions and labels
- **Font Family**: System sans-serif stack for optimal readability

### Component Patterns
- **Cards**: Rectangular containers with subtle shadows for content grouping
- **Buttons**: Primary (blue), Secondary (purple outline), Ghost (text-only) variations
- **Navigation**: Horizontal menu with smooth hover transitions
- **Project Grid**: Responsive masonry layout for showcasing projects
- **Modals**: Centered overlays for detailed project views
- **Forms**: Minimal input fields with focus states and validation indicators
